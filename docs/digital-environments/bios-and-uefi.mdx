## UEFI vs BIOS
`-` UEFI has more graphical and interactive look compared to BIOS. UEFI stores all data about initialisation and startup <br />
`-` UEFI has faster boot times <br />
`-` UEFI supports secure boot, which secures the device from booting from unauthorised applications <br />
`-` UEFI runs in 32bit or 64bit mode, whereas BIOS runs in 16bit mode <br />
`-` BIOS based systems use MBR partitions, while UEFI ones use GPT ones <br />
`-` On a GPT disk, you can use more than four partitions on each disk <br />
`-` GPT is also required for disks larger than 2 TB <br />

|                            | MBR                  | GPT                    |
|----------------------------|----------------------|------------------------|
| Maximum Partition Capacity | 2TB                  | 8.4ZB                  |
| Maximum Partition Number   | 4 primary partitions | 128 primary partitions |
| Firmware support           | BIOS                 | UEFI                   |
| Operating system support   | Windows 7 and older  | Windows 8/9.1/10       |
