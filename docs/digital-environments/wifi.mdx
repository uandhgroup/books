## Mac and IP

### MAC
<p className="grey">Media Access Control</p>

`-` A unique identifier assigned to a network interface controller for use as a network address in communications within a network segment. 12 digit hexadecimal number. Hex base 16.

### IP
<p className="grey">Internet Protocol</p>

`-` A numerical label such as 192.0.2.1 that is connected to a computer network that uses the Internet Protocol for communication. V4 or V6. <br />
`-` Logical Address. Can and will be changed per connection. Static or dynamic <br />
`-` Issued by server, router or other network device. V6 will replace V4 as it provides many more addresses than V4 <br />
