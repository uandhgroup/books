## Tasks of a Level 2 Support


`-` Finding solutions based on previous cases <br />
`-` Closing a reassigning jobs <br />
`-` Handling configuration issues <br />
`-` Manage software installations and hardware repair <br />
`-` Review information and training available for first level support <br />
`-` Monitor available tools for issues (network) <br />
`-` Collect information and escalate to L3 support <br />
`-` Hardware repair <br />
`-` Physical Visit <br />
`-` Respond to messages relating to hardware problems or system issues. <br />
