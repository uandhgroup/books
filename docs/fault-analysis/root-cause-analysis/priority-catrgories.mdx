<p className="grey">There are different priorities for different issues.</p>

## Low priority

`-` Mouse not working <br />
`-` Monitor damage <br />
`-` USB slot not working <br />
`-` Keyboard key missing <br />
`-` Slow Computer <br />

## Medium priority

`-` One computer not working <br />
`-` Sound not working <br />
`-` Network cable broken for a device <br />
`-` Software not working <br />

## High priority

`-` Internet outage <br />
`-` Power outage <br />
`-` Overload <br />
`-` Risk of data corruption <br />
`-` Server down <br />
