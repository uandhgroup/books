`-` <code className="greennumbercode">Input</code> <br />
`└──` This is data inputted. This data is preparing to be processed. <br />

`-` <code className="greennumbercode">Storage</code>  <br />
`└──`This is the recording and retention of data on an appropriate format: <br />
`ㅤ` `├──` <code className="greennumbercode">Create/store</code>  - Retain data records for future use or compliance <br />
`ㅤ` `└──` <code className="greennumbercode">Organise</code> - Restructure and rank data in a specific order <br />


`-` <code className="greennumbercode">Processing</code>  <br /> 
`└──` Transforming data into meaningful output: <br />
`ㅤ` `├──` <code className="greennumbercode">Analyse</code> - Business/digital insight through search queries/criteria <br /> 
`ㅤ` `├──` <code className="greennumbercode">Update</code> - Ensuring data records are up to date <br /> 
`ㅤ` `├──` <code className="greennumbercode">Remove</code> - Removal of data entries where appropriate <br /> 
`ㅤ` `└──` <code className="greennumbercode">Integrate</code> - Integrate different sets of information together <br /> 


`-` <code className="greennumbercode">Output</code>  <br />
`└──` Data generated by the information system: <br />
`ㅤ` `├──` <code className="greennumbercode">Read/search</code> - Identify and find specific information <br />
`ㅤ` `└──` <code className="greennumbercode">Insight</code> - Gain from processing to support decisions <br />


`-` <code className="greennumbercode">Feedback loop</code>  <br />
`└──` A system structure that allows output to influence future input
