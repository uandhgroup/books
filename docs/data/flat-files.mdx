Flat files are where you only need to find one place to access the file. 


## Advantages
`-` All records are stored in one place <br />
`-`Easy to understand and configure using various standard office applications. <br />
`-` It is an excellent option for small databases <br />
`-` It requires less hardware and software components <br />
`-` Can be used with no special software <br />


## Disadvantages
`-` Flat file databases are harder to update <br />
`-` Harder to change data formats <br />
`-` It is a poor option for databases in terms of complex queries <br />
