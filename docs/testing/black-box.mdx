<p className="grey">A software testing method in which the internal structure/design/implementation of the item being tested is not known to the tester.</p>

`-` Mostly done by software testers <br />
`-` It is functional test of the software <br />
`-` No knowledge of programming is required <br />
`-` It is the behaviour testing of the software <br />
`-` Can be done by trial and error ways and methods <br />
